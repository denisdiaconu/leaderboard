# Leaderboard

> I built a leaderboard website that displays scores submitted by different players.
## Built With

- JavaScript
- HTML
- CSS



## Getting Started

To get a local copy up and running follow these simple example steps on the command line.

```bash

  # Clone the project from it's github repo.

  $ git clone https://github.com/denisdiaconu/leaderboard.git

  # Change directory to the root of project

  $ cd leaderboard

  # Install all dev dependencies

  $ npm install
    
  $ npm start

```
